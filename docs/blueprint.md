# **App Name**: BiciSecure

## Core Features:

- Authentication Flow: User authentication with 'ciclista' role by default; admin role through Firebase Custom Claims.
- Homepage: Editable homepage sections via admin panel. Includes a public search bar.
- Public Search: Search public bycicle from bikes in app database.
- Bike Registration: Form for cyclists and bikeshops to register bikes with serial number, make, model, color, photos, and ownership documents.
- Cyclist Dashboard: Display user's registered bikes with options to view details, report theft, or transfer ownership; displays bike status.
- Bike Details: Show bike photos, ownership, location, status (stolen/safe), and theft reporting button.
- Theft Report: A feature that marks a bicycle as stolen/recovered
- Home edit: Admin functionality that permits edit text or image for the Homepage sections.

## Style Guidelines:

- Primary color: Cyan (#00FFFF) to reflect trust, security, and a modern feel.
- Background color: Light Cyan (#E0FFFF) for a clean and airy backdrop.
- Accent color: Blue (#00BFFF) to complement cyan and reinforce the theme of security.
- Body and headline font: 'Inter' sans-serif font for clear and accessible readability.
- Use a 'shield with check' icon for the logo to represent safety and verification.
- ShadCN UI components within Tailwind CSS for a responsive, modern UI.
- Subtle transition effects for form elements and feedback messages.
